房地產投資報酬率計算器

房地產投資報酬率計算器使用說明

請我喝珍奶！

概述

本計算器幫助您評估全額或貸款買房的年化報酬率（CAGR），並與台北市CPI及其他資產（比特幣、S&P 500、0050、黃金、債券）進行比較。所有現實成本（如交易稅、仲介費、維護費用、出租成本、裝潢成本）均納入計算，確保結果貼近實際。

使用步驟

  1. 選擇投資類型：全額買房或貸款買房。
  2. 選擇是否出租：若選擇「是」，輸入年租金收入；若選擇「否」，租金收入為0。
  3. 輸入基本資訊：
    • 購買金額：房屋購入價格（預設10,000,000新台幣）。
    • 當前房價：當前房屋市場價值（預設12,000,000新台幣）。
    • 年租金收入：出租房屋的年租金（預設240,000新台幣）。
    • 貸款金額（貸款買房）：借貸金額（預設7,000,000新台幣）。
    • 貸款期限（貸款買房）：貸款還款年限（預設20年）。
    • 年利率（貸款買房）：貸款年利率（預設2.5%）。
    • 投資開始與結束年份：選擇投資期間（預設2014-2024）。
  4. 輸入成本資訊：
    • 買房交易稅：如契稅（預設2%購買金額，200,000新台幣）。
    • 賣房交易稅：如增值稅（預設2%當前房價，240,000新台幣）。
    • 買房仲介費：買房時的仲介費用（預設1.5%購買金額，150,000新台幣）。
    • 賣房仲介費：賣房時的仲介費用（預設1.5%當前房價，180,000新台幣）。
    • 房屋維護費用：持有期間的物業管理或維修費（預設每年10,000新台幣）。
    • 裝潢成本：房屋裝潢費用（預設500,000新台幣）。
    • 出租相關成本：如空置率、維修費（預設年租金的10%，24,000新台幣/年）。
    • 選擇「有」或「無」，若選擇「有」但未輸入金額，使用預設值；選擇「無」則費用為0。
  5. 點擊「計算報酬率」按鈕，生成報告，顯示年化報酬率、CPI比較、其他資產回報及圖表。
  6. 點擊「下載報告」按鈕，將報告與圖表生成為PDF（需先計算報酬率）。

計算邏輯

全額買房

  • 初始投資：購買金額 + 買房交易稅 + 買房仲介費 + 裝潢成本
  • 總回報：
    • 房價增值 = 當前房價 - 賣房交易稅 - 賣房仲介費 - 購買金額
    • 租金收入 = (年租金收入 - 出租相關成本) × 年數
    • 總回報 = 房價增值 + 租金收入 - (房屋維護費用 × 年數)
  • 年化報酬率（CAGR）： 
    CAGR = [(當前房價 - 賣房交易稅 - 賣房仲介費 + (年租金收入 - 出租相關成本) × 年數 - (房屋維護費用 × 年數)) / (購買金額 + 買房交易稅 + 買房仲介費 + 裝潢成本)]^(1/年數) - 1

貸款買房

  • 初始投資：購買金額 - 貸款金額 + 買房交易稅 + 買房仲介費 + 裝潢成本
  • 總回報：
    • 房價增值 = 當前房價 - 賣房交易稅 - 賣房仲介費 - 購買金額
    • 租金收入 = (年租金收入 - 出租相關成本) × 年數
    • 貸款總成本 = 月付金額 × 貸款期限 × 12
    • 月付金額： 
      月付金額 = [貸款金額 × (年利率 / 12)] / [1 - (1 + 年利率 / 12)^(-貸款期限 × 12)]
    • 總回報 = 房價增值 + 租金收入 - 貸款總成本 - (房屋維護費用 × 年數)
  • 年化報酬率（CAGR）： 
    CAGR = [(當前房價 - 賣房交易稅 - 賣房仲介費 + (年租金收入 - 出租相關成本) × 年數 - 貸款總成本 - (房屋維護費用 × 年數)) / (購買金額 - 貸款金額 + 買房交易稅 + 買房仲介費 + 裝潢成本)]^(1/年數) - 1
    若總回報為負，顯示「虧損」。

台北市CPI年化增長率

  • 使用1998-2024年『真實』CPI數據。
  • CPI CAGR = (結束年CPI / 開始年CPI)^(1/年數) - 1

其他資產回報

  • 假設年化報酬率：比特幣25%、S&P 500 10%、0050 8%、黃金5%、債券3%
  • 回報 = 初始投資 × [(1 + 年化報酬率)^年數 - 1]

注意事項

  • 數據來源：CPI與資產報酬率為假設值，實際應用需使用官方數據。
  • 成本假設：預設值基於台北市房地產市場，實際情況可能不同。
  • 貸款計算：假設固定利率，未考慮變動利率或提前還款。
  • PDF生成：若下載PDF失敗，可稍後重試或檢查瀏覽器設置。
