SQL CREATE - 建立資料表、資料庫或其他物件

語法說明

CREATE 用於建立資料庫中的新物件，例如資料表、資料庫、索引等。最常見的是建立資料表，其基本語法如下：

CREATE TABLE table_name (
    column1 datatype constraints,
    column2 datatype constraints,
    ...
);

範例參數：

  • table_name：資料表名稱
  • datatype：資料型態（如INT、VARCHAR、DATE）
  • constraints：約束條件（如PRIMARY KEY、NOT NULL）

實務案例

假設我們要建立一個名為 employees 的員工資料表，包含員工ID、姓名與入職日期。語法如下：

CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    hire_date DATE
);

執行後，產生的資料表結構如下：

欄位名稱 資料型態 約束條件
id INT PRIMARY KEY
name VARCHAR(50) NOT NULL
hire_date DATE -

這個資料表可用來儲存員工的基本資訊，id 是主鍵，確保每筆資料唯一，name 不可為空。
