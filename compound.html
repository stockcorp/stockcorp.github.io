歡迎體驗「複利計算機」 - 您的投資成長最佳助手！

想知道投資能成長多少？「複利計算機」，提供「入門」與「進階定期定額」兩種模式，幫您計算複利收益並提供分析表格。快來試試吧！

入門計算機使用說明

輸入欄位
- 初期本金：初始投入金額，例如 20,000 元。
- 計息期數 (n)：投資年數，例如 20 年。
- 收益率 (%)：年化收益率，例如 5%。
計算邏輯
總額 = 初期本金 × (1 + 收益率)^計息期數，每年收益基於前一年總額計算。
輸出結果
- 總收益與最終金額。
- 逐年表格顯示年份、收益、總額與累計收益率。

進階定期定額計算機使用說明

輸入欄位
- 初期本金：初始投入，例如 20,000 元。
- 每月投資金額：每月定投，例如 5,000 元。
- 投資期數：投資時間，例如 5 年。
- 利率 (%)：年化或月化利率，例如 5%。
- 複利方式：年複利、月複利等，例如年複利。
計算邏輯
每月追加投資並按複利方式計算收益。
輸出結果
- 總收益、最終金額、總投資額。
- 可切換年/月表格顯示本金、收益與最終金額。

月複利計算公式
公式
futureValue = previousValue * (1 + monthlyRate) + monthlyInvestment

變數說明
- futureValue：當前月份的未來價值（最終金額）。
- previousValue：上個月的總額（本金加利息）。
- monthlyRate：月利率，通常是年化利率 (annualRate) 除以12，即 monthlyRate = annualRate / 12。
- monthlyInvestment：每個月追加的投資金額。

原理
月複利的特點是每個月計算一次利息，並將該利息加入本金，作為下個月計算利息的基礎。公式分為兩部分：
- previousValue * (1 + monthlyRate)：計算上個月總額的利息並加到本金上。
- + monthlyInvestment：加上當月新投入的金額。
這種方式模擬了現實中每月定投的情況，例如定期存款或基金定投，且利息每月結算並滾入本金。
逐步計算範例
假設：初期本金20,000，每月投資5,000，5%年利率，月複利，計算前3個月：
初始條件：
- annualRate = 0.05（5%）
- monthlyRate = 0.05 / 12 ≈ 0.0041667
- previousValue = 20,000
第1個月：
- futureValue = 20,000 * (1 + 0.0041667) + 5,000
- = 20,000 * 1.0041667 + 5,000
- = 20,083.33 + 5,000 ≈ 25,083.33
第2個月：
- previousValue = 25,083.33
- futureValue = 25,083.33 * (1 + 0.0041667) + 5,000
- = 25,083.33 * 1.0041667 + 5,000
- = 25,187.81 + 5,000 ≈ 30,187.81
第3個月：
- previousValue = 30,187.81
- futureValue = 30,187.81 * (1 + 0.0041667) + 5,000
- = 30,187.81 * 1.0041667 + 5,000
- = 30,313.59 + 5,000 ≈ 35,313.59

年複利計算公式
公式
currentValue = (previousValue + monthlyInvestment * yearMonths) * (1 + annualRate)

變數說明
- currentValue：當前年份的總額（最終金額）。
- previousValue：上一年的總額。
- monthlyInvestment：每個月追加的投資金額。
- yearMonths：當年的月數（通常為12，若是最後一年可能少於12）。
- annualRate：年化利率，例如5%則為0.05。

原理
年複利的特點是每年計算一次利息，將全年累積的本金和利息作為下一年計算的基礎。公式分為兩部分：
- previousValue + monthlyInvestment * yearMonths：計算當年結束時的總本金（上一年總額加上當年每月投入）。
- * (1 + annualRate)：對全年總本金計算一次年利息，並加入本金。
這模擬了每年結算一次利息的情況，例如某些年度定投產品。
逐步計算範例
假設：初期本金20,000，每月投資5,000，5%年利率，年複利，計算前2年：
初始條件：
- annualRate = 0.05
- yearMonths = 12
- previousValue = 20,000
第1年：
- currentValue = (20,000 + 5,000 * 12) * (1 + 0.05)
- = (20,000 + 60,000) * 1.05
- = 80,000 * 1.05 = 84,000
第2年：
- previousValue = 84,000
- currentValue = (84,000 + 5,000 * 12) * (1 + 0.05)
- = (84,000 + 60,000) * 1.05
- = 144,000 * 1.05 = 151,200
